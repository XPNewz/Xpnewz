Earth SMP

Earth SMP is a Minecraft multiplayer survival server that recreates the earth in 1:1000 scale in game. This server shows earth geography, all nations, biomes and many man made structures and still evolving. Earth SMP allows freedom to every player to create any structure wherever they want and can explore the whole world and interact with the environment.

Know more information here:

How to Join Earth SMP server

  1. Server IP Address:
    • You can find the server IP address on the Earth SMP website or Discord server. It might change depending on which version of the server you're playing.
  2. Whitewashing
      Some Earth SMP servers ask players to be whitelisted before they can join. This helps create a grown-up and committed group of players.
  3. Playable Version:
      Earth SMP works with the newest Minecraft Java Edition, but you should check what version the server needs. Many servers are available for Bedrock also.
  4. Rules:
      Most Earth SMP servers have strict rules against destroying things, cheating, and being mean to keep everyone happy.

Main Features

Real-World Earth Map:

  • The server employs a created map that mirrors Earth's landscape such as mountains, rivers, oceans, and deserts.
  • The ratio stands at 1:1000, with one Minecraft block equal to one kilometer in actual life.
  • The map draws from real-world geographic information to ensure the highest possible accuracy.
  • Earth SMP servers rely on unique world generation tools like Terra++ or Dynmap to build a Minecraft world that copies Earth's geography.

Economy System

  • Many Earth SMPs boast in-game economies often using add-ons like EssentialsX, ShopGUI, or ChestShop.
  • Players can purchase and sell land, exchange resources, and run businesses.
  • Some servers have economies driven by players where supply and demand affect in-game costs.

Custom Features & Plugins

  • Vehicles: A few servers add planes, cars, or boats through Movecraft or CustomVehicles.
  • Geopolitical Systems: Players can stake out land, create nations, and take part in diplomacy or conflict.
  • Teleportation & Travel: A few servers let you use /warp, /home, or /tpa, while others push you to use boats and Elytras to explore.

Steps to Join server

  1. Open Minecraft
  2. Go to Multiplayer option (Java), go to server option (Bedrock)
  3. Click on Add server
  4. Enter name of server
  5. Enter IP address and port
  6. and Join the server
